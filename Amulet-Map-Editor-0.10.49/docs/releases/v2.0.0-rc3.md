# Amulet Map Editor v2.0.0-rc3

Release date: 2026-03-03

## Highlights
- New top-down lasso selection mode with live preview while selecting.
- Export-time custom block remapping with an editable local remap table:
  - `custom_block_export_remap.json`
  - auto-discovery of custom namespaces in selected chunks before export.
- Reduced warning spam by filtering repeated custom-block translation warnings.
- Additional hardening for Bedrock world open/preflight and `.mcworld` handling.

## Release Check Command
- `python scripts/release_smoke_test.py`

## Notes
- This is a release candidate for the `v2.0.0` line.
- Manual validation is still recommended for:
  - Bedrock and Java restore workflows,
  - export remap behavior on worlds containing add-on/custom namespaces.
