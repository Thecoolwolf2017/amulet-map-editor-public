# Amulet Map Editor v2.0.0-rc4

Release date: 2026-03-03

## Highlights
- Fix for export remap preview crash caused by a wx window ID lifetime issue.
- `Preview Remap` can now be opened repeatedly without hitting:
  - `DecIdRefCount(): id count already 0`
- Verification pass completed in the release environment:
  - `python -m py_compile .../remap_ui.py`
  - `python -m compileall -q amulet_map_editor tests`
  - `python scripts/release_smoke_test.py`
  - `python -m pytest -q tests/test_export_remap.py`

## Notes
- This is a release candidate for the `v2.0.0` line.
- Recommended for users seeing export remap preview crashes on `v2.0.0-rc3`.
