# Amulet Map Editor
### World Migration & Conversion

Private Windows binary distribution for Amulet Map Editor.

## Download
- Get the latest Windows zip from this repository's Releases page.

## Run
1. Download the release zip (`Amulet-v*-Windows-x64.zip`).
2. Extract it to a folder on your computer.
3. Run `amulet_app.exe`.

## Repository Layout
- `Amulet-Map-Editor-0.10.49/`: main source tree, docs, tests, and build config.
- `.github/`: repository-level workflows and metadata.

## Current Build Output
- Main executable: `Amulet-Map-Editor-0.10.49/dist/Amulet/amulet_app.exe`
- Release zip + checksums: `Amulet-Map-Editor-0.10.49/dist/`

## Distribution Policy
- Binary-only distribution.
- No `.whl` or `.tar.gz` Python package artifacts are published.
